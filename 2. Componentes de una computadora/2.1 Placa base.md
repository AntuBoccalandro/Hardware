# 2.1 La placa madre

### ¿Qué es?

La placa base se puede llamar de diferentes maneras, las cuales son: placa base, motherboard o placa madre. Esto es importante saberlo ya que se la suele llamar de diferentes maneras como las mencionadas anteriormente, pero todas se refieren al mismo componente.

La placa base es el componente de hardware que conecta a todos los componentes como el procesador, las memorias o el botón de encendido entre sí, para que un componente pueda conectarse y actuar sobre el otro. Es básicamente la base donde interconectamos todos los componentes para que funcionen como un todo. 

### ¿Cuál es su función?

La placa base es el componente de hardware que conecta a todos los componentes como el procesador, las memorias o el botón de encendido entre sí, para que un componente pueda conectarse y actuar sobre el otro. Es básicamente la base donde interconectamos todos los componentes para que funcionen como un todo. 

La placa base se compone de una PCB  (Print Circuit Board o placa de circuito impreso) la Figura 1.1 muestra diferentes tipos de PCBs. Es una placa o lámina de un grosor aproximado de entre 1 a 3mm. En esta placa se encuentran impresos diferentes circuitos que interconectan a los diferentes de los cuales se compone la placa en sí. Además se encuentran impresos buses de alta velocidad de transferencia de datos para conectar componentes externos a la placa.

### Tipos

Existen diferentes tipos de placas madres según el uso que se le dará a la misma. Al igual que con muchos elementos, como los computadores vistos en la sección 1° que se detallan los diferentes tipos de ordenadores también existen diferentes tipos de placas bases y demás componentes que puede incluir una computadora como procesadores, targetas gráfica o memoria RAM.

Es importante saber que la placa base va directamente conectada con el hardware y supone un punto imporante a considerar para la compatibilidad de los diferentes componentes de un ordenador. Placas bases que no contengan características suficientes para un hardware a conectar supondrán una traba a la hora de utilizar ese componente, esta falla puede deberse a una incopatibilidad de la placa base con el componente a conectar o simplemente la placa base no admite ese tipo de hardware.

**Escritorio**: las placas bases de computadores de escritorio no tienen características a destacar, cuentan con componentes muy básicos que para la mayoría de ordenadores de oficina servirán y no supodrán una limitación, esta se verá agravada a la hora que se quiera actualizar el hardware de la computadora a uno más potente que requiera de ciertas especificaciones mayores. Este tipo de placa suelen ser de las más económicas. Por último, la estética de este tipo de placas no es su punto fuerte.  

**Gaming**: Las placas bases dedicadas al gaming o videojuegos se encuentran orientadas a una utilidad y sobre todo estética de este nicho. Este tipo de placa base contienen componentes de calidad para poder tener compatibilidad con hardware potente que requiera de una placa base con un mínimo de características pero sin llegar al extremo, es decir, cuenta con componentes que soportaran cualquier tipo de hardware pero sin llegar a ser muy constosas. 

**Workstations**: como hablamos anteriormente las workstations son computadoras pensadas para profesionales que en su trabajo requieran de gran cantidad de potencia. Las placas bases dedicadas a este tipo de computadoras cuentan con características que permiten aprovechar al máximo este tipo de hardware. Contienen capacidad para la conexión de varias targetas gráficas (aunque muchas otras placas también), la posibilidad de conección de muchos módulos de RAM así como muchos puertos de entrada y salida para poder utilizar gran cantidad de dispositivos sin necesidad de utilizar extensores o hubs USB (estos puertos son de gran velocidad para poder transferir gran cantidad de datos de manera rápida), la cantidad de puertos suele estar asociada a que este tipo de profesionales en ciertas áreas se utilizan muchos dispositivos, como cámaras, monitores, grabadoras, etc.

**Servidores**: las placas bases de servidores cuentan con un diseño y uso muy diferente al que tendría un ordenador de escritorio. Estas cuentan, al igual que las workstations, la posibilidad de conección de gran cantidad de componentes, sobre todo RAM. Este tipo además cuenta en su gran mayoría de veces con dos slots de coneccción de CPUs que permiten conectar dos procesadores a una placa, algo que en placas bases para ordenadores de sobremesa no profesionales no es posible. 

### Tamaños

Existen tres tamaños de de placas bases, esto es así para poder utilizarlas en disintos tipos de espacios sean estos más espaciosos o más reducidos.

![](https://external-content.duckduckgo.com/iu/?u=https%3A%2F%2Fwww.wikiversus.com%2Finformatica%2Fplacas-base%2Fformatos%2Fimg%2Featx-vs-atx-vs-micro-atx-vs-mini-itx-featured_hu2c267064193ff3765f2bca895011c1a3_446507_1400x787_fill_q85_lanczos_center.jpg&f=1&nofb=1&ipt=a2ffeec942931f4d7b2f6880dc50826446d1c6e89f0c5fb210f0352fa19fb20d&ipo=images)

**E-ATX**: su sigla remarca el nombre completo de Extended-ATX, este tamaño de placas bases es el más grande que existe. Para utilizarlas es necesario de un gabinete o carcasa de dichas dimensiones. Las dimensiones de 305 x 330mm de estas placas hacen que se catalogen como E-ATX. Suelen admitir hardware muy potente pero esto también sucede con el resto de tamaños. 

**ATX**: es el formato más antiguo de placas bases. Las medidas de este tamaño son de 305 x 244mm. La mayoría de placas bases del mercado suelen ser de este tamaño. Encontrarás muy variadas y una gran compatibilidad de hardware así como de gabinetes.

**Micro-ATX** este formato de placa base mide 244 x 244mm, placa que necesita de dicho tipo de gabinete de computadora.

**Mini-ATX**: este formato mide 170 x 170mm y es el tamaño de placa base más pequeño que hay. Su uso suele estar orientado a ordenadores que quieran ocupar poco espacio. Para este tipo de placas es necesario tener un gabinete de cualquier tamaño, aunque por obvias razones están diseñadas para utilizarse en gabinetes muy peueñas o personalizados. Este tipo de placas bases suelen tener varios "pisos" esto para poder comprimir todos los componentes que tendría una placa base de tamaño completo pero en uno mucho más reducido. 

### ¿Cómo funciona? 

El funcionamiento de las placas bases es muy sencillo, como ya dijimos se trata de una placa de circuito impreso, mediante todos los componentes que la componen tales como: capacitores, resistencias, reguladores, trasnsitores, chipsets, etc. Estos componentes se encuentras interconectados por medio de buses o pistas que a groso modo serían los "cables" que conectan todos los componentes de la placa para que funcionen en conjunto. 

### Componentes externos

La placa base se compone de muchos componentes y los que podríamos catalogar como externos serían las tapas que se encargan de la estética y protección de la placa ya que hay muchos otros componentes que son los que realmente interesan que si bien los catalogamos como internos ni bien podrían ser externos también. 

![](../src/Sección%202/Placa%20base/S%202%20Disipadores.jpg)

**Protectores y disipadores**: los protectores de una placa base suelen ser paneles de plástico o metal que se encargan de tapar ciertas zonas de la placa base así como también ofrecer disipación en zonas que son propensas a sobrecalentar o durante largas jornadas de exigencia puedan llegar a tomar temperatura. como ya dijimos los protectores pueden ser puramente estéticos u ofrecer una cierta funcionalidad. Los disipadores que puede incluir una placa base pueden ser *activas* o *pasivas*. La refrigeración o disipación activa es aquella que refirgera al/los componentes por medio de algún medio móvil, ejemplos son: ventiladores, turbinas, líquida. La refrigeración pasiva consiste en disipar el calor mediante la transferencia del mismo, ejemplo son: tapas de metales altamente conductores del calor como el cobre o alumnio.

### Componentes internos

Los componentes internos de una placa base son el: 

### Socket

![](../src/Sección%202/Placa%20base/S%202%20Socket(2).jpg)

**Zócalo o socket:** es donde se conecta el procesador. Es un conector (como un USB,por ejemplo, es muy diferente) que se conecta al procesador mediante una serie de pines. Existen dos tipos de zócalos: los que tienen los pines de conexión en la placa base o los que tienen el conector hembra en la placa base, pues el procesador será quien cuente con los pines. ¿Por qué existen estos dos tipos? Estos dos principales tipos de zócalos (en inglés socket) van a depender de la marca del procesador. Actualmente existen dos: Intel y AMD (dentro de las PC). Estos dos fabricantes de procesadores tienen dos tipos de procesadores diferentes en cuanto a la conexión nos referimos, es por ello que encontramos dos tipos de sockets. Mientras que AMD apuesta por colocar los pines en el procesador Intel apuesta por los pines en el socket de la placa base. Hay puntos a tener en cuenta que son importantes a la hora de elegir una placa base como lo es la compatibilidad de sockets. Como ya vimos existen dos principales tipos de conexión entre el procesador y la placa base. Pero es importante saber que también dentro de cada marca de procesadores existen distintos sockets, por lo cuál podemos tener distintos procesadores de la misma marca pero que dependiendo de la generación o modelo estos no sean compatibles con el socket de la placa base. Esta información de compatibilidad entre procesador y placa base la podemos encontrar en las especificaciones del mismo, dirá algo así como: socket: LGA 775. Es importante que el socket del procesador coincida con el de la placa base. ¿Por qué existen diferentes tipos de sockets? Ni bien van desarrollando nuevas tecnologías y van evolucionando los procesadores los sockets se van quedando atrás, es por eso que tienen que ir evolucionando también ya que si no las velocidades altas de transferencias de datos que tienen los procesadores no son aprovechados por el socket.

### VRM

![](../src/Sección%202/Placa%20base/S%202%20VRM.jpg)

**VRM:** El VRM (Voltage Regulator Module o módulo regulador de voltaje) es el encargado de rectificar la corriente que ingresa al procesador ya quela energía que ingresadesde la PSU (Power Supply Unit o Unidad suplidora de energía) o simplemente fuente de alimentación es corriente continua, pero tiene algo de micro variaciones que pueden afectar al correcto funcionamiento del procesador, ya que es un componente muy delicado. El módulo regulador de voltaje como su propio nombre lo indica se encarga de regular el voltaje. Convierte los, por ejemplo, 12V que ingresan a la placa base a la energía que necesite el procesador en dicho momento determinado, además de hacer un proceso de filtrado o rectificado a la corriente, para que sea más estable como se puede ver en la Figura 1.1. Este componente es muy importante ya que sin él sería mucho más inestable el funcionamiento del ordenador y permite el cuidado de los componentes contra las bajas o subidas de tensión que puedan ocasionarse.

### Slots de RAM

![](../src/Sección%202/Placa%20base/S%202%20Slot_RAM(1).jpg)

**Ranuras o slots de memoria RAM:**es donde va conectada la memoria RAM. Este tipo de memoria tiene una forma rectangular de un grosor aproximado de 1cm. Debajo del módulo de memoria RAM encontramos una serie de buses de conexión. Estos, conectados al slot o ranura de la placa base permite la interconexión entre el módulo de memoria, la placa base y el procesador. Ya que internamente hay una serie de circuitos y buses impresos que le permiten a la memoria RAM enviar y recibir datos del procesador y de la unidad de almacenamiento principal.

### Slots M.2

![](../src/Sección%202/Placa%20base/S%202%20Slot_M2.jpg)

**Slot o ranura M2:** esta ranura nos permite conectar unidades de almacenamiento de tipo SSD.M2. Las ranuras M2 son básicamente un área rectangular en la que en un lado de los más cortos de dicho rectángulo encontramos unos contactos que mediante conexiones internas de la placa base se conectan a la memoria RAM y a demás componentes. Del otro lado hay un canal para atornillar y ajustar dicha unidad de almacenamiento.

### Puertos de entrada y salida

![](../src/Sección%202/Placa%20base/S%202%20Puertos_E-S.jpg)

**Puertos de entrada y salida:**los puertos de salida de la placa base son básicamente los puertos que nos permiten conectar periféricos al ordenador, tales como teclados, mouses o pantallas. Entre los puertos de salida encontramos a puertos USB, HDMI, Displayport, jack de auriculares o puertos ethernet. 

### Conectores SATA

![](../src/Sección%202/Placa%20base/S%202%20SATA.jpg)

**Conectores SATA:** los conectores SATA (Serial Advanced Technologies Attachment) son los conectores que nos permiten conectar unidades de almacenamiento. Dependiendo del modelo de la placa base estas tendrán más conectores o menos. Es importante aclarar que según la generación del conector o puerto SATA estos podrán transferir más o menos datos.

### Conector ATX

![](../src/Sección%202/Placa%20base/S%202%2024_pines.jpg)

**Conector ATX:** el conector ATX nos permite suministrarle energía a la placa base. Este se compone de 24 pines, también es conocido como conector de 24 pines o ATX.

### Conector de 8 pines o CPU

![](../src/Sección%202/Placa%20base/S%202%208_pines.jpg)

**Conector de 8 pines o CPU:** este conector es el encargado de suministrar la energía necesaria al procesador, que como ya vimos es necesario que pase antes por el VRM. Entonces tenemos al conector de 8 pines que es exclusivo para el procesador y luego tenemos al conector de 24 pines que se encarga de suministrar energía a los USBs, la memoria RAM, entre otros componentes excepto las unidades de almacenamiento y la tarjeta gráfica.

### Conector ventilador

**Conector del ventilador:** es donde se conecta el disipador que refrigera el procesador. En caso de ser una refrigeración activa se utilizará este conector, en caso contrario no se utilizará.

### DAC

**Chip de sonido o DAC:** este módulo también es conocido como DAC (Digital Analog Converter) se encarga de la conversión del sonido en formato digital a analógico, esto es necesario ya que las señales que producen las computadoras son digitales y todos los dispositivos de audio utilizan sistemas analógicos para transmitir el sonido y hacerlo llegar hasta nuestro oído. Los DAC incorporados en las placas suelen ser bastantes malos en cuanto a la calidad de dicha conversión pero es cierto que esta calidad variará según la gama de la placa base.

### Conector PCI/PCI-E

![](../src/Sección%202/Placa%20base/S%202%20Slot_PCIe.jpg)

**Conectores PCI:** existen dos tipos de conectores PCI en una placa base moderna. Primero tenemos el viejo PCI que cuenta con velocidades inferiores de transferencias en comparación al PCI Express. Estos conectores tienen la forma de una ranura en la cuál se pueden conectar componentes aparte de la placa en sí. Como tarjetas gráficas, de red o de sonido. Luego tenemos al PCI Express. Este conector ofrece una conexión mucho más directa con los diferentes componentes de la placa. Además de contar con velocidades de transferencias de datos mucho mayores a la del viejo PCI. Dentro de los conectores PCI Express podemos encontrar distintas clasificaciones y tamaños, como lo son el PCI Express x1, x4, x16.

### BIOS

**BIOS:** la BIOS (Basic Input Output System) es un módulo o memoria ROM (Read Only Memory) que se encuentra integrada en la placa y es la que contiene todo el código necesario para el arranque o booteo del computador. Básicamente lo que permite la BIOS es la comunicación inicial entre el hardware y el software. Iniciar el monitor, permitir la escritura y control del PC a través del teclado y mouse.

### Pila de la BIOS

![](../src/Sección%202/Placa%20base/S%202%20Pila_BIOS.jpg)

**Pila o batería:** es una pequeña batería que le suministra energía a la BIOS para que esta no pierda las configuraciones del PC así como también permitir que funcione el RTC (Real Time Clock).

### Nortbridge

**Northbridge:** este chip permite la conexión entre el procesador, la memoria RAM y la tarjeta gráfica. Algunas placas más viejas cuentan con este chip pero en los procesadores actuales ya viene integrado dentro del mismo.

### Southbridge

**Southbridge:** también conocido como ICH (I/O Controller Hub). Este chip se encarga de controlar las comunicaciones con los dispositivos periféricos de entrada y salida. Así como también la comunicación entre el procesador y los conectores SATA, PCI y PCI Express. Además de la comunicación con las tarjetas de sonido, ethernet y USBs.

### Reloj

**Reloj:** el reloj o RTC (Real Time Clock) de una placa base se encarga de ir actualizando la hora del computador. Esta es la manera en la cuál puedes encender luego de la noche el PC y este tendrá la hora correspondiente a la mañana y no la última vez que lo hayas apagado.

### Chipsets

![](../src/Sección%202/Placa%20base/S%202%20Chips.jpg)

**Chipsets:** son el grupo de chips que tiene incorporado la placa base. Los chipsets que vienen con la placa base tienen que ser compatibles con el procesador que se va a aconsejar. Los chipsets son una serie de chips que controlarán frecuencia y voltajes del procesador.

### Botón de encendido/apagado/reseteo

**Botón de encendido/apagado/reseteo:** este botón permite el encendido, el apagado y reinicio del computador. Es útil para hacer pruebas acerca de la correcta conexión de los componentes a la placa o simplemente por la utilidad de algún momento en el que el PC falle.

### Agujeros de tornillos de conexión del ventilador

**Agujeros de conexión de disipadores:** estos agujeros similares a los de un tornillo permiten anclar el disipador a la placa base (ya sea un disipador activo o pasivo).

### Agujeros de tornillos de la placa base

**Agujeros de la placa base:** estos agujeros sirven para atornillar la placa base al gabinete del computador o a otra superficie.

Ya hemos hecho un recorrido por cada componente básico de cualquier placa base. A partir de los cuales cada placa dependiendo de la gama incluirá más o menos componentes así como de mejor o peor calidad.
